# 校园绿植养护 WebGIS 管理系统技术说明概要

## 项目定位

系统面向校园绿化管理与日常巡查场景，使用 WebGIS 方式集中展示绿植空间分布、健康状态、养护记录与问题上报信息，为校园后勤绿化维护和学生志愿巡查提供统一平台。

## 技术路线

- 前端：HTML5、CSS3、原生 JavaScript、Canvas 图表
- 后端：Python Flask
- 演示存储：localStorage
- 生产迁移：PostgreSQL + PostGIS
- 空间能力：点位标注、热力图、坐标采集、空间聚合视图

## 核心模块

1. 用户认证与权限控制
2. WebGIS 地图与绿植点位展示
3. 绿植档案管理
4. 养护打卡与审核
5. 问题上报与处理流转
6. 数据统计看板
7. 管理员后台
8. 校园植物图鉴与个人中心
